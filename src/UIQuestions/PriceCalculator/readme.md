# Functional Requirements
-> Build a three column layout where first column is the product quantity.
-> The second column is the price of the product.
-> The third column tells how much amount is needed to buy that product.
-> The amount should update as soon as price or quantity is changed.
-> We can add more products by clicking of a button at the bottom.
-> At the end there is a field which notifies the total amount required to buy all the products.
